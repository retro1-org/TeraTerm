Theme Editor

テーマはTera Termの背景、文字の色、または、両方を一括して扱う仕組み。
各タブで設定を行う。
[OK] で動作中のTera Termに反映され、[Cancel] で反映されずに破棄される。
preview/file tab
テーマの一時的な設定(preview),テーマファイルの読み込みと書き出し
Backgroud tab
BG(背景画像)に関する設定を行う
color tab
文字の色に関する設定を行う
テーマファイルについて

preview/file tab

preview
テーマの一時的な設定(preview)、やり直し(undo)を行う
[preview]
設定したテーマを一時的に設定(preview)する
[undo]
変更したテーマをダイアログを開く前の状態に戻す
file
テーマファイルからテーマを読み込み、テーマを保存する
[load]
テーマファイルを読み込む
[save]
設定中のテーマをファイルに書き出す
BGテーマ、Colorテーマを書き出すかを選択できる

Backgroud tab(背景テーマ)

*注意* 背景画像を表示するため、文字の背景は無効化される。

Tera Term の背景画像は次のように指定する。

テーマファイルについて(Eterm風半透明ウィンドウ)

iniファイル内のキーについてのメモ

仕様が固まったら消す
BGPictureTone
背景画像(Dest)にデスクトップ画像(Src1)どれぐらいブレンドするか
BGSrc1Alpha = 255 - BGPictureTone
0=デスクトップ画像(Src1)が100%
255=背景画像(Dest)が100%
BGFadeColor
Src2の色
BGFadeColor = BGSrc2Color
BGFadeTone
DestとSrc1の合成画像に単色(Src2)をどれぐらいブレンドするか
BGSrc2Alpha = 255 - BGFadeTone
0=単色(Src2)が100%
255=DestとSrc1の合成画像が100%

color tab(カラーテーマ)

*注意* 背景画像を表示しているとき、文字の背景は無効化される。

カラーテーマはデフォルト色に上書きして色が設定される。
デフォルト色はTERATERM.INI で設定される。
iniファイル内で設定しない色はデフォルト色となる。

このタブでは次のことができる

  • 色の設定
  • デフォルトの色に戻す

色の設定はリストで色を選んで右クリック(orダブルクリック)

今のところ commctrlのカラーピッカーを使用。
